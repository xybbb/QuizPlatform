方式一：上传 ZIP 压缩包（推荐，支持题目配图）

压缩包内包含一个 Excel/CSV 数据文件和题目配图文件，配图将自动关联到对应题目。

压缩包结构示例： 
questions.zip
├── 题目数据.xlsx          ← 数据文件（必须）
├── image1.png             ← 配图文件（可选）
├── image2.jpg             ← 配图文件（可选）
└── ...
方式二：直接上传 Excel/CSV 文件（不含配图）

也支持直接上传 .xlsx / .xls / .csv 文件，但不支持配图。

数据文件必须包含以下列（列名必须完全一致）：

列名 是否必填 说明 示例
分类名称 必填 题目所属分类，不存在会自动创建 数学
分组名称 可选 题目所属分组，不存在会自动创建 第一章
题型 必填 仅为以下四种之一：single / multiple / judge / fill single
题干 必填 题目的完整内容 1 + 1 = ?
题目图片 可选 ZIP 压缩包中配图的文件名（如 image1.png） image1.png
选项 条件必填 单选/多选必填，用竖线 | 分隔各选项；判断/填空留空 A.1|B.2|C.3|D.4
正确答案 必填 单选题：选项字母（A/B/C/D）；多选：用逗号分隔（A,C）；判断：对/错；填空：答案文本（多个用|分隔） B
解析 可选 答案解析文本 根据加法运算法则，1+1=2
难度 可选 1-5 的整数，1 最简单，5 最难，默认为 1 1
